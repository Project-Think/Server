## 首页

### 1. 返回标签列表

- 请求方式: `get`
- 请求地址: `/tags`
- 返回数据示例:
  ```json
  {
    "code": 200,
    "message": "ok",
    "data": ["前端", "后端", "大数据"]
  }
  ```

### 2. 返回推荐的文章列表

- 请求方式: `get`
- 请求地址：`/articleList/recommend`
- 返回数据示例
  ```json
  {
    "code": 200,
    "message": "ok",
    "data": []
  }
  ```

### 3. 返回热门的文章列表

- 请求方式: `get`
- 请求地址：`/articleList/hot`
- 返回数据示例
  ```json
  {
    "code": 200,
    "message": "ok",
    "data": []
  }
  ```

### 4. 返回最新的文章列表

- 请求方式: `get`
- 请求地址：`/articleList/new`
- 返回数据示例
  ```json
  {
    "code": 200,
    "message": "ok",
    "data": []
  }
  ```

### 5. 根据标签返回文章列表

- 请求方式: `get`
- 请求地址：`/articleList/byTag`
- 参数类型：`query`

| 参数 | 是否必须 | 类型   | 说明 |
| ---- | -------- | ------ | ---- |
| tag  | Y        | 字符串 | 标签 |

- 请求示例
  `/articleList/byTag?tag=前端`
- 返回数据示例
  ```json
  {
    "code": 200,
    "message": "ok",
    "data": []
  }
  ```

## 详情

### 1、根据 id 返回文章对象

- 请求方式: `get`
- 请求地址：`/article`
- 参数类型：`query`

| 参数 | 是否必须 | 类型 | 说明 |
| ---- | -------- | ---- | ---- |
| id   | Y        | 数字 |      |

- 请求示例
  `/article?id=123`
- 返回数据示例
  ```json
  {
    "code": 200,
    "message": "ok",
    "data": {}
  }
  ```
- 备注
  该文章的阅读量会加 1

- 返回数据示例
