## 对象说明

### 用户对象

| id       |      | name  | 名字       |
| -------- | ---- | ----- | ---------- |
| password | 密码 | token | 安全字符串 |



### 文章对象

| id        |                    | title       | 文章标题                         |
| --------- | ------------------ | ----------- | -------------------------------- |
| user.id   | 文章作者的id       | user.name   | 文章作者名字                     |
| user.url  | 文章作者的主页链接 | type        | 文章类型：article 文章 news 资讯 |
| tags      | 标签列表           | isRecommend | 是否为推荐文章                   |
| likeCount | 点赞数             | viewCount   | 阅读数                           |
| date      | 发布时间戳         | content     | 文章内容                         |
| comments  | 评论列表           | url         | 文章链接                         |
| imgUrl    | 文章图片链接       | isOrigin    | 是否为原创，否则就是转载文章     |



## 首页

### 1. 返回标签列表

- 请求方式: `get`
- 请求地址: `/tags`
- 返回数据示例
  ```json
  {
    "code": 200,
    "message": "ok",
    "data": ["前端", "后端", "大数据"]
  }
  ```

### 2. 返回为你推荐的文章列表

- 请求方式: `get`
- 请求地址：`/articleList/recommend`
- 返回数据示例
  ```json
  {
    "code": 200,
    "message": "ok",
    "data": []
  }
  // data 是一个若干个文章对象的数组
  ```

### 3. 返回近期热门的文章列表

- 请求方式: `get`

- 请求地址：

  - 日热门：`/articleList/hot?type=day`
  - 周热门：`/articleList/hot?type=week`
  - 月热门：`/articleList/hot?type=mouth`

- 返回数据示例
  ```json
  {
    "code": 200,
    "message": "ok",
    "data": []
  }
  // data 是一个若干个文章对象的数组
  ```

### 4. 返回最新的文章列表

- 请求方式: `get`
- 请求地址：`/articleList/new`
- 返回数据示例
  ```json
  {
    "code": 200,
    "message": "ok",
    "data": []
  }
  // data 是一个若干个文章对象的数组
  ```

### 5. 根据标签返回文章列表

- 请求方式: `get`
- 请求地址：`/articleList/byTag`
- 参数类型：`query`

| 参数 | 是否必须 | 类型   | 说明 |
| ---- | -------- | ------ | ---- |
| tag  | Y        | 字符串 | 标签 |

- 请求示例
  `/articleList/byTag?tag=前端`
- 返回数据示例
  ```json
  {
    "code": 200,
    "message": "ok",
    "data": []
  }
  // data 是一个若干个文章对象的数组
  ```

## 详情

### 1、根据 id 返回文章对象

- 请求方式: `get`
- 请求地址：`/article`
- 参数类型：`query`

| 参数 | 是否必须 | 类型 | 说明 |
| ---- | -------- | ---- | ---- |
| id   | Y        | 数字 |      |

- 请求示例

- 返回数据示例
  ```json

    "code": 200,
  ```
